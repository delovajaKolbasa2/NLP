# NLP

This repository is for NLP course 2025–26.

## Branches

* **main** – contains materials and code for Assignment 1.
* **main-second** – branch created for Assignment 2 development.
* **main-third** – branch created for Assignment 3 development.

## Reproducibility

Python version: 3.11

### Setup

1. In terminal

   ```bash
   cd "pathname_of_desired_folder"
   ```

2. Clone this repository:

   ```bash
   git clone https://github.com/delovajaKolbasa2/NLP.git
   ```

3. Open repository folder

   ```bash
   cd NLP
   ```

4. Initialize virtual environment

   ```bash
   python -m venv .venv
   ```

5. Activate virtual environment

   **Windows**

   ```bash
   .venv\Scripts\activate
   ```

   **Mac/Linux**

   ```bash
   source .venv/bin/activate
   ```

6. Install requirements

   ```bash
   pip install -r requirements.txt
   ```

7. Install Jupyter (optional)

   ```bash
   pip install jupyterlab
   ```

8. Run the code (requires Jupyter)

   ```bash
   jupyter lab
   ```

### Working with branches

To switch to the Assignment 3 branch after cloning:

```bash
git checkout main-third
```
